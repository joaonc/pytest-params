============
Installation
============


Stable release
--------------

To install pytest params, run this command in your terminal:

.. code-block:: console

    $ pip install pytest_params

This is the preferred method to install pytest params, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for pytest params can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/joaonc/pytest_params

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/joaonc/pytest_params/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/joaonc/pytest_params
.. _tarball: https://github.com/joaonc/pytest_params/tarball/master
